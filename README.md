
# Calculator

This is a basic calculator android app




## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear
- Delete
- Parenthesis
- Decimal Point(.)
## Methodology
- Infix to Postfix conversion
- Postfix Evaluation
## Extra Features
- Lottie Animation
